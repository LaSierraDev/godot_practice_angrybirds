# 🎮 Godot Practice — Angry Birds Clone

A small practice project built in **Godot 4.6.1**, focused on developing a **physics-based projectile game** inspired by Angry Birds.

## 🧩 Implemented Features

### 🎮 Gameplay
- Slingshot launching mechanics  
- Vector-based trajectory calculation and launch direction  
- Launch limit per level  
- Health/damage system: enemies take damage and die when boxes fall on them  
- Object destruction and chain reactions  
- Score system based on enemy defeat and destruction  

### 🧱 Physics & Collisions
- Physics body and area handling  
- Group system usage  
- Vector-based movement logic for projectiles and objects  

### 👾 Enemies
- Damage calculation based on collisions  
- Interaction with falling boxes and other objects  

### 🖥️ UI & Scenes
- Score and remaining launches display  
- Options menu navigation  
- Scene switching  
- Node instancing and removal  
- `CanvasLayer` for proper UI layout  
- Responsive scaling for multiple resolutions  
- Custom UI themes  

### 🎨 Environment
- Destructible block structures  
- Backgrounds and parallax layers  
- Audio system  
- On-screen node activation/deactivation optimization  
- `AnimationPlayer` for object and enemy animations  

### 💻 Code Architecture (GDScript)
- Clean, readable, and scalable code  
- Modular structure with reusable components  
- Clear separation of responsibilities  
- Global variable management  

+ Patterns:
  + Signal Hub  
  + Singleton (Autoloads)  

- Signals  
- Custom signals  
- Timers  
- Tweens  
- Inherited scenes  

### 🎥 Camera
- Camera follow system to track the projectile  
- Smooth camera movement during launches  

## 🛠️ Technologies
- Godot Engine 4.6.1  
- GDScript  
- Git (Version control)
